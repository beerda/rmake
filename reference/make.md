# Run `make` in the system

This function executes the `make` command to rebuild all dependencies
according to the `Makefile` generated by
[`makefile()`](https://beerda.github.io/rmake/reference/makefile.md).

## Usage

``` r
make(..., .stdout = "", .stderr = "", .stdin = "")
```

## Arguments

- ...:

  Command-line arguments passed to the `make` command (see `?make` in
  your shell for details)

- .stdout:

  Where to direct standard output; see
  [`base::system2()`](https://rdrr.io/r/base/system2.html).

- .stderr:

  Where to direct standard error; see
  [`base::system2()`](https://rdrr.io/r/base/system2.html).

- .stdin:

  Where to get standard input; see
  [`base::system2()`](https://rdrr.io/r/base/system2.html)

## Value

Exit status of the command; see
[`base::system2()`](https://rdrr.io/r/base/system2.html) for details.

## See also

[`makefile()`](https://beerda.github.io/rmake/reference/makefile.md),
[`rmakeSkeleton()`](https://beerda.github.io/rmake/reference/rmakeSkeleton.md)
